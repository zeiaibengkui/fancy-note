# Fancy Note

A fancy note software with plentiful features.

## Features

- Home Page
- Mindmap
- Dark mode
- Multi-language
- Easy-using
- Totally free
- Humble programmer (I guess...)

## Project Setup

```sh
pnpm install
```

### Development

```sh
pnpm dev
```

### Compile for Production

```sh
pnpm build
```
